# mkpass
パスワード作成ツール

## 特徴
指定された文字を必ず1文字以上使いランダムな文字列を生成する

## 使い方
mkpass.py [options] 桁数

## options
* -d, --digits: 数値を使う
* -l, --lower: アルファベット小文字を使う
* -u, --upper: アルファベット大文字を使う
* -s, --string: アルファベットを使う(-l -u と等価)
* -m,  --mark 記号: 指定された記号を使う
